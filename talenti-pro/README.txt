

Descrição do projeto, instalação, tecnologias e instruções.


1 - Tecnologias do projeto:
  - Java (versão 17)
  - WildFly (Servidor de aplicação versão 27.0.0.Final)
  - PrimeFaces (versão 14.0.8)
  - PostgreSQL (versão 18)
  - Hibernate (versão 6.1.5.Final)
  - JSF (versão 4.0.1)

Apenas precisamos instalar o JDK 17, o PostgreSQL 18 e o WildFly 27.

Após a instalação realizar as configurações definidas nos tutoriais
além da configuração de ambiente de variáveis, instalações como Eclipse IDE,
importe do projeto do Git e ajustes do ambiente.
